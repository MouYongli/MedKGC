快速开始

快速入门指南，帮助您在几分钟内开始使用 MedKGC。

  • 安装
  • 使用方法
  • API 文档

安装

按照以下步骤安装 Project MedKGC：

  1. 克隆仓库：git clone https://github.com/yourusername/MedKGC.git
  2. 进入项目目录：cd MedKGC
  3. 安装依赖项：npm install
  4. 启动项目：npm start

使用方法

如何使用 Project MedKGC 进行医学知识图谱的构建和查询。

  • 数据导入：npm run import-data
  • 启动推理引擎：npm run start-inference
  • 访问可视化界面：http://localhost:3000

API 文档

详细的 API 接口说明，帮助开发者集成和扩展 MedKGC 的功能。

  • 获取实体
  • 获取关系
  • 添加实体
  • 添加关系

获取实体

获取知识图谱中的所有实体。

GET /api/entities

获取关系

获取知识图谱中的所有关系。

GET /api/relations

添加实体

向知识图谱中添加新的实体。

POST /api/entities
Content-Type: application/json

{
    "name": "实体名称",
    "type": "实体类型"
}

添加关系

在知识图谱中创建实体之间的关系。

POST /api/relations
Content-Type: application/json

{
    "from": "实体ID1",
    "to": "实体ID2",
    "relation": "关系类型"
}

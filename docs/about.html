关于 Project MedKGC

Project MedKGC 是一个专注于构建和维护医学知识图谱的开源项目。我们的目标是通过整合多源医学数据，提供一个统一、可扩展且易于使用的平台，促进医学研究和临床应用。

我们的使命

通过先进的知识图谱技术，推动医疗信息的智能化和互联化，提升医疗服务的质量和效率。

我们的团队

我们的团队由来自数据科学、医学、信息技术等多个领域的专家组成，致力于跨学科合作，共同推动医学知识图谱的发展。
